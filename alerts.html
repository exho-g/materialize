Alerts are great for getting attention in a page as it provides contextual feedback messages for typical user actions. Check Rinebeck's source codes here.

check_circle Oh! What a beautiful default alert
check_circle Oh! What a beautiful primary colored alert
warning Oh! What a beautiful danger colored alert close 
              Copied!
              content_copy
              
<div class="col s12">
    <div class="alert">
        <i class="material-icons">check_circle</i>
        <span>Oh! What a beautiful default alert</span>
    </div>

    <div class="alert primary">
        <i class="material-icons">check_circle</i>
        <span>Oh! What a beautiful primary colored alert</span>
    </div>

    <div class="alert outlined thin danger">
      <i class="material-icons">warning</i>
      <span>Oh! What a beautiful danger colored alert</span>
      <a href="#!" class="waves-effect btn-soft danger alert-close tooltipped" data-position="bottom"
        data-tooltip="Close">
        <i class="material-icons">close</i>
      </a>
    </div>
</div>

Actions

You can also add an action button inside. It is recommended to use the .btn-flat as it follows the material guidelines.

check_circle Oh! What a beautiful alert with action Click Here
check_circle Oh! What a beautiful primary colored alert Click Here
warning Oh! What a beautiful danger colored warning with close trigger close 
              Copied!
              content_copy
              
<div class="col s12">
    <div class="alert">
        <i class="material-icons">check_circle</i>
        <span>Oh! What a beautiful alert with action</span>
        <a href="#!" class="btn-flat waves-effect">Click Here</a>
    </div>
    <div class="alert primary">
        <i class="material-icons">check_circle</i>
        <span>Oh! What a beautiful primary colored alert</span>
        <a href="#!" class="btn-flat waves-effect">Click Here</a>
    </div>
    <div class="alert warning">
      <i class="material-icons">warning</i>
      <span>Oh! What a beautiful danger colored warning with close trigger</span>
      <a href="#!" class="waves-effect btn-flat alert-close tooltipped" data-position="bottom"
        data-tooltip="Close">
        <i class="material-icons">close</i>
      </a>
    </div>
</div>
