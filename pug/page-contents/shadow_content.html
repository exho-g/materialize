In material design, everything should have a certain z-depth that determines how far raised or close to the page the element is.

You can easily apply this shadow effect by adding a class="z-depth-2" to an HTML tag. Alternatively you can extend any of these shadows with Sass using @extend .z-depth-2. A z-depth-0 can be used to remove shadows from elements that have z-depths by default. Please note that shadows are not visible on dark mode and is recommended to change background color brightness depending on the depth level.


    <div class="col s12 m2">
      <p class="z-depth-1 ondark-z-depth-0 ondark-teal ondark-darken-3">z-depth-1 ondark-z-depth-0 ondark-teal ondark-darken-3</p>
    </div>
    <div class="col s12 m2">

    </div>
