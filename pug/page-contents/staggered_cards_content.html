Staggered Cards makes a Pinterest like layout.

Basic

To get started, just enclose your cards with .staggered-grid class and each cards enclosed with .item class. It is recommended to use .card or .card-panel as root elements inside .item

Initialization

You can run the following code snippet to initialize the staggered card's.


  M.resizeStaggeredCards();

Masonry Cards

Use this if you prefer the non-javascript version.

To use, just add a container with a .masonry-cards class, and then add your .card or .card-panel inside. Credit's to MichaelLNorth for this helper. To see it in action, checkout his codepen.

Please note that this will not work if the card has a .col class.
